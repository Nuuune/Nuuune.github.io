当参数为对象时，此数组里的每一项都是这个对象的引用
当遍历改变此数组里的对象时，所有数组项同时改变

```javascript
const list = Array(20).fill({ url: "xxx", id=-1});
list.forEach((curr, index) => {
  curr.id = index;
}); 

list.forEach((curr) => {
  console.log(curr.id);    //所有输出都为19
});
```
